# Медицинская страховка для путешествий

## Зачем нужна страховка

Медицинская страховка за границей критически важна по нескольким причинам:

1. **Медицина за границей дорогая** - один визит к врачу может стоить $100-300, госпитализация - $1000-5000+
2. **Визовые требования** - многие страны требуют страховку для въезда (Шенген, США)
3. **Трансляция и репатриация** - эвакуация в Россию стоит десятки тысяч долларов
4. **Юридическая помощь** - помощь с документами при госпитализации за рубежом

## Что должно быть в полисе

### Минимальный набор (для Вьетнама и Азии):
- Медицинские расходы (от $10,000)
- Амбулаторное лечение (без госпитализации)
- Стоматология (обычно до $500-1000, только острая боль)
- Телефон горячей линии 24/7

### Опционально (рекомендуется):
- Отмена поездки (при болезни/несчастном случае до вылета)
- Задержка рейса (компенсация отеля и питания)
- Потеря багажа
- Спорт (обычно не входит; нужна опция для активного отдыха)

## Франшиза и лимиты

- **Франшиза** - сумма, которую платите сами. Либо 0 франшиза (дороже), либо фиксированная (например, $50 за визит)
- **Лимит на один случай** - максимальная выплата за одно событие (рекомендуется от $10,000 для Азии, $30,000+ для Европы/США)
- **Лимит на сутки** - ограничение по количеству дней лечения (обычно 60-90 дней в год)

## Стоимость страховки

Примерные ставки для человека 30-40 лет:
- Азия (Вьетнам, Таиланд): $1-2 в день
- Европа (Шенген): $2-3 в день
- Мировое покрытие: $3-5 в день

Проверяйте страховые компании в реестре ЦБ РФ: https://www.cbr.ru/registries/

## Проверенные страховщики (Россия)

- **Alpha Insurance** - один из крупнейших, хорошая репутация
- **Ingosstrakh** - работает с 1947 года, широкое покрытие
- **Ренессанс Страхование** - конкурентные цены
- **Согласие** - быстрые выплаты по отзывам

Проверяйте условия именно для вашей страны назначения - покрытие может отличаться.

## Что делать при страховом случае

1. Позвонить в службу ассистанса (телефон на полисе) **ДО** обращения к врачу (кроме жизнеугрожающих случаев)
2. Сохранить все чеки, справки, выписки из больницы
3. По возвращении подать заявление в страховую с документами
4. Срок рассмотрения обычно 15-30 дней
